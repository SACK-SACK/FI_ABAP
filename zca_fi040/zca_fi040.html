include ZCA_FI040_TOP. " 전역변수 선언
include ZCA_FI040_SCR. " Selection Screen
include ZCA_FI040_CLS. " Local Class ( Event Handler Method )


*&---------------------------------------------------------------------*
* 구현 관련 Include
*&---------------------------------------------------------------------*
* 프로그램 코딩하면서 자동으로 생성되기 때문에 Include를 직접 들어가지는 않는다.
include ZCA_FI040_PBO. " Process Before Output
include ZCA_FI040_PAI. " Process After Input
include ZCA_FI040_F01. " Form Subroutines

INITIALIZATION.

AT SELECTION-SCREEN.

AT SELECTION-SCREEN OUTPUT.
  LOOP AT SCREEN.
    IF screen-group1 = 'FIX'. " 위의 MODIF ID fix와 일치
      screen-input = 0.      " 입력 불가
      screen-output = 1.     " 화면 표시
      MODIFY SCREEN.
    ENDIF.
  ENDLOOP.

START-OF-SELECTION.

  PERFORM select_data.

* 출력용 데이터를 화면에 List로 출력하기 위해 0100 화면을 호출
  PERFORM display_data. " 작성 후 스크린 생성
